## agentcli-go v0.2.2

Release date: 2026-02-24

### Highlights

- Added monorepo scaffold mode:
  - `agentcli new --in-existing-module <name>`
  - no nested `go.mod`, parent module import path is auto-resolved
- Standalone scaffold now runs `go mod tidy` automatically, so `go.sum` is generated on creation.
- Added cross-repo wrapper preset:
  - `agentcli add command --preset task-replay-emit-wrapper replay-emit`
  - supports `--repo`, optional `--task`, repeatable `--env KEY=VALUE`
- Unified loop API/CLI summary shape via harness envelope (`internal/loopapi.RunSummary`).

### Quickstart

```bash
go install github.com/gh-xj/agentcli-go/cmd/agentcli@v0.2.2

# standalone project
agentcli new --module example.com/mycli mycli

# monorepo project (inside existing Go module)
agentcli new --dir ./tools --in-existing-module replay-cli
```

### Notes

- Binary name is `agentcli`; repository/module name is `agentcli-go`.
