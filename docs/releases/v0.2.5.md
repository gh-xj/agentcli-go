## agentcli-go v0.2.5

Release date: 2026-02-24

### Highlights

- Added first-class migration command for existing shell script repos:
  - `agentcli migrate --source ./scripts --mode safe --dry-run`
  - `agentcli migrate --source ./scripts --mode safe --apply`
- Added dual migration modes:
  - `safe` (default): generates a parallel `agentcli-migrated` workspace
  - `in-place`: writes migration outputs into the current repo
- Added migration engine (`internal/migrate`) with a low-cognitive-load pipeline:
  - scan (`bash`/`sh` detection + risk signals)
  - plan (`auto`, `wrapper`, `manual`)
  - generate (wrapper command stubs + migration docs/artifacts)
- Added migration contract checks in CI/local gate:
  - `schemas/migrate-plan.schema.json`
  - `schemas/migrate-report.schema.json`
  - schema fixtures under `testdata/contracts/migrate-*`

### Quickstart

```bash
go install github.com/gh-xj/agentcli-go/cmd/agentcli@v0.2.5
agentcli --help
agentcli migrate --source ./scripts --mode safe --dry-run
agentcli migrate --source ./scripts --mode safe --apply
```

### Notes

- v1 migration target is `bash` + POSIX `sh`.
- Generated migration artifacts are under `docs/migration/` in the selected output root.
- If proxy propagation lags, retry install with:
  `GOPROXY=direct go install github.com/gh-xj/agentcli-go/cmd/agentcli@v0.2.5`.
