## agentcli-go v0.2.0

First public release of `agentcli-go` as a deterministic, AI-agent-friendly Go CLI framework.

### Highlights

- Scaffold CLI (`agentcli`):
  - `agentcli new`
  - `agentcli add command`
  - `agentcli doctor --json`
- Contract-first verification model:
  - schema-backed JSON output checks
  - positive and negative fixtures in CI
- Runtime modules:
  - `cobrax` for standardized Cobra command runtime and exit-code handling
  - `configx` for deterministic config precedence (`Defaults < File < Env < Flags`)
- Quality gates:
  - canonical CI contract via `task ci`
  - local aggregate verification via `task verify`
- OSS onboarding improvements:
  - `LICENSE`, `CONTRIBUTING`, `CODE_OF_CONDUCT`, `SECURITY`, `CHANGELOG`
  - issue templates, PR template, CI workflow
  - docs-site skeleton and runnable examples

### Quickstart

```bash
go install github.com/gh-xj/agentcli-go/cmd/agentcli@v0.2.0
agentcli new --module example.com/mycli mycli
agentcli add command --dir ./mycli sync-data
agentcli doctor --dir ./mycli --json
```

### Notes

- This release emphasizes deterministic scaffolding and verification contracts for human + AI-agent workflows.
