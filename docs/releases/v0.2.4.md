## agentcli-go v0.2.4

Release date: 2026-02-24

### Highlights

- Added `agentcli new --minimal` mode to generate a minimal runnable scaffold.
- Added `task-replay-orchestrator` preset for external repo task orchestration with:
  - repeated `--env KEY=VALUE`
  - `--timeout`
  - `--timeout-hook`
- Added release safety hardening:
  - `task release:gate` now includes `go run ./cmd/agentcli --help`
  - core-only compile path with `go build -tags agentcli_core ./cmd/agentcli`
- Refactored loop command surface for lower maintenance:
  - removed legacy adapter wrappers in `cmd/agentcli/loop.go`
  - tests now validate parser/profile behavior directly in `tools/harness/commands`
- Removed deprecated preset `task-replay-emit-wrapper` (single recommended path: `task-replay-orchestrator`).

### Quickstart

```bash
go install github.com/gh-xj/agentcli-go/cmd/agentcli@v0.2.4
agentcli new --minimal demo-cli
agentcli add command --preset task-replay-orchestrator replay-orchestrate
```

### Notes

- Monorepo recommendation remains: prefer `agentcli new --in-existing-module`.
- If proxy propagation lags, retry install with direct proxy:
  `GOPROXY=direct go install github.com/gh-xj/agentcli-go/cmd/agentcli@v0.2.4`.
